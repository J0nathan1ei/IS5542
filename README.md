# CityU Maison Storefront

A static fashion ecommerce showcase for the CityU Maison brand.

## Local Preview

Run a static server from this folder:

```bash
python3 -m http.server 4173
```

Then open:

```text
http://localhost:4173
```

## Publish With GitHub Pages

1. Create an empty GitHub repository, for example `cityu-maison-storefront`.
2. Connect this local project to it:

```bash
git remote add origin https://github.com/YOUR_USERNAME/cityu-maison-storefront.git
git push -u origin main
```

3. In GitHub, open `Settings` -> `Pages`.
4. Set `Source` to `Deploy from a branch`.
5. Choose `main` and `/root`, then save.

Your site will be available at:

```text
https://YOUR_USERNAME.github.io/cityu-maison-storefront/
```
